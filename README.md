# AI 영어 문제 분석기

React + Vite + PWA 기반의 영어 학습 지원 애플리케이션입니다.

## 폴더 구조
- `English-learning-assistant/`: 프론트엔드(개발/빌드 대상)
- `supabase/`: Supabase Edge Functions / 마이그레이션
- `test_image/`: 로컬 테스트용 이미지

## 로컬 개발
```bash
cd English-learning-assistant
npm install
npm run dev
```

## 배포 (Netlify)
Netlify는 레포 루트의 `netlify.toml` 설정에 따라 `English-learning-assistant`를 빌드/배포합니다.
