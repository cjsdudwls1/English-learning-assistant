## 1. 모델별/기능별 비용 요약 (원화 기준)

가장 궁금해하시는 **"기능 1회 실행 시 얼마가 드는지"**를 정리한 핵심 요약표입니다. (환율 1,400원 기준)

| :--- | :---: | :---: | :--- |
| **이미지 분석**<br>(1장, 1~2문제) | **약 1원** | **약 5원** | 가장 빈번한 사용 패턴 |
| **이미지 분석**<br>(1장, 7문제 꽉 참) | **약 2.5원** | **약 17원** | 출력이 많아 비용 증가 |
| **예시 문장 생성** | **약 0.25원** | **약 1.5원** | - |

## 3. 상세 비용 분석 (토큰량 기반)

각 기능이 실제로 사용하는 토큰 양과 이를 기반으로 계산된 상세 비용입니다.

### 3.1 이미지 분석 (Analyze Image) - 실측 데이터

가장 핵심적인 기능입니다. **DB 실측 데이터 기반**으로 작성되었습니다.

| 시나리오 | 평균 토큰 (입력 / 출력) | 비용 (Lite) | 비용 (Gemini 3) | 데이터 출처 |
| :--- | :---: | :---: | :---: | :--- |
| **기본 분석** (1장, 1문제) | 5,300 / 400 | **0.97원** | **5.39원** | DB 실측 (4건 평균) |
| **상세 분석** (1장, 7문제) | 5,700 / 3,000 | **2.48원** | **16.59원** | DB 실측 (1건) |
| **다중 페이지** (3장, 10문제) | 16,000 / 4,000 | **4.48원** | **28.00원** | 추정 |

> **계산 공식**: (입력 토큰 × 입력 단가) + (출력 토큰 × 출력 단가)
> - Lite: (입력 × 0.00014원) + (출력 × 0.00056원)
> - Gemini 3: (입력 × 0.0007원) + (출력 × 0.0042원)

## 6. 제언

1.  **현재 상태 유지 권장**: 현재 적용된 **Gemini 2.5 Flash Lite**는 성능 대비 비용 효율이 매우 뛰어납니다. (이미지 1장 분석에 **1원 미만**)
2.  **고성능 모델 도입 가능성**: 향후 분석 정확도를 높이기 위해 **Gemini 3 Flash Preview**로 전환하더라도 비용 부담은 크지 않습니다. (학원 규모에서도 월 1만원 미만)
3.  **이미지 리사이즈**: 현재 원본 이미지를 전송 중이나, 비용이 워낙 저렴하므로 굳이 리사이즈를 하여 화질(인식률)을 낮출 필요가 없습니다. **현행 유지**를 추천합니다.
