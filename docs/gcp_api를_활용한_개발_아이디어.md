# GCP API를 활용한 English Learning Assistant 고도화 개발 아이디어

현재 구축된 '오프라인 시험지 스캔 및 AI 채점' 기능을 바탕으로 적용할 수 있는 강력한 GCP 기술 기반 확장 아이디어들입니다.

---

## 1. 🗣️ "스피킹 & 리스닝" 무한 확장 (Cloud TTS / Cloud STT)
앱의 초점을 '읽기/풀기'에서 '듣기/말하기' 영역까지 넓히는 기능입니다.

*   **나만의 지문 오디오북 (Cloud Text-to-Speech)**
    *   **기능**: Document AI가 추출한 시험지 지문 텍스트를 원어민 음성으로 즉각 생성.
    *   **효과**: 버튼 하나로 지문 듣기 무한 반복 및 스크립트 기반(Time-stamped) 강조 표시 기능 구현 가능.
*   **서술형 답안 "혀"로 풀기 (Cloud Speech-to-Text)**
    *   **기능**: 학생이 스마트폰에 대고 영어 문장을 말하면 이를 텍스트로 변환하여 문제 정답과 대조해 채점.
    *   **효과**: 쓰기의 번거로움을 줄이고 참여도 증대. 발음의 정확도(Shadowing) 훈련에 응용 가능.

## 2. ✍️ "첨삭 전문 원어민 과외 선생님" (Cloud Natural Language & Gemini)
단순한 정오답 판별을 넘어, 원인 분석과 교정에 초점을 맞춥니다.

*   **서술형 오답 '핀셋' 분석기**
    *   **기능**: 학생의 오답(`Is → Are` 등)을 수집하여 "수일치 오류", "스펠링 오류" 등으로 자동 태깅.
    *   **효과**: Natural Language API로 문장 성분을 분해하여 학생이 자주 틀리는 문법 요소 정확히 파악.
*   **유사 문제/오답 노트 무한 생성 (Gemini)**
    *   **기능**: 학생이 틀린 특정 문법(예: 관계대명사)이 포함된 100% 매칭되는 새로운 영작/객관식 문제를 실시간 생성.
*   **작문(Essay) 첨삭 봇**
    *   **기능**: 자유 영작 사진 업로드 시, 문법 오류를 지적하고 "원어민식의 더 자연스러운 문장"으로 교정본 제시.

## 3. 🎯 "개인 맞춤형 취약점 공략" (Vertex AI & BigQuery)
데이터베이스(Supabase)에 누적되는 데이터를 활용한 맞춤형 학습 관리입니다.

*   **AI 약점 진단 리포트**
    *   **기능**: 누적된 오답 데이터를 바탕으로 취약점 패턴을 텍스트 및 그래프 형태로 요약(예: "과거 완료 시제 오답률 80%").
    *   **효과**: 학생별/학급별 주간 혹은 월간 리포트 자동 발행 서비스 도입.
*   **내 오답노트로 만든 나만의 단어장**
    *   **기능**: 틀린 지문에 사용된 고난이도 영단어를 문서에서 자동 추출(Entity Extraction).
    *   **효과**: 원어민 발음(TTS)이 결합된 '깜빡이 단어장' 및 단어 퀴즈 자동 생성.

## 4. 📸 UX/UI 및 편의성의 극대화 (Cloud Vision API)
앱 사용의 스트레스를 줄이고 잔존율(Retention)을 높입니다.

*   **스마트 크롭 & 책상 쓰레기 자동 필터링 (Cloud Vision API)**
    *   **기능**: 사진 속 사물/배경을 인식해 시험지가 아닌 부분(손가락, 책상, 지우개)을 자동 블러(Blur) 처리 또는 잘라내기.
    *   **효과**: Document AI 처리 전 깨끗한 이미지를 확보하여 오인식 사전 방지.
*   **진행 상태 동기화 및 뱃지 시스템 (게이미피케이션)**
    *   **기능**: AI 분석 데이터를 기반으로 "연속 5일 학습 달성", "관계대명사 마스터" 등의 업적 배지 부여.
    *   **효과**: 수동적인 시험지 촬영을 능동적인 게임 퀘스트처럼 변화시킴.
